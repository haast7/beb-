# Certificados

Esta pasta contém as imagens dos certificados da Haast.

## Arquivos necessários:

- `meta-ads.png` - Certificado Meta Ads
- `google-ads.png` - Certificado Google Ads  
- `rd-station.png` - Certificado RD Station

## Especificações das imagens:

- **Formato**: PNG (recomendado) ou JPG
- **Dimensões**: 60x60px (ou maior, será redimensionado automaticamente)
- **Fundo**: Transparente ou branco
- **Qualidade**: Alta resolução para melhor visualização

## Como adicionar:

1. Substitua os arquivos placeholder pelos certificados reais
2. Mantenha os mesmos nomes de arquivo
3. As imagens aparecerão automaticamente na seção de oferta

## Estrutura atual:

```
public/certificates/
├── meta-ads.png      (placeholder)
├── google-ads.png    (placeholder)
├── rd-station.png    (placeholder)
└── README.md         (este arquivo)
```
